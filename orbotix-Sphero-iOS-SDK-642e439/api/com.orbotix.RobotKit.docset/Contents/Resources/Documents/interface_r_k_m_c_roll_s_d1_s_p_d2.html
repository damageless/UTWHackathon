Detailed Description

This command will cause the robot to roll at the given heading. The system speed 2 variable will be used with this command. It can be set using the RKMCSPD2 command. The robot will continue rolling until a roll command with a speed of 0.0 is issued. A delay will be automatically added after this command with the value specified in the system delay 1 variable. This can be set using the MCSD1 command. Allowed values: heading 0-359
