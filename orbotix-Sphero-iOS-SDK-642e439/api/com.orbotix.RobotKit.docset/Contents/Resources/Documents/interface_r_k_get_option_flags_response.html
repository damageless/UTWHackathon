Detailed Description

Bit 0: Set to prevent Sphero from going to sleep when placed in the charger and connected over Bluetooth. Bit 1: Set to enable Vector Drive, that is, when Sphero is stopped and a new roll command is issued it achieves the heading before moving along it. Bit 2: Set to disable self-leveling when Sphero is inserted into the charger. Bit 3: Set to force the tail LED always on. Bit 4: Set to enable motion timeouts
