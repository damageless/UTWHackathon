Detailed Description

This class will discover, advertise, connect to and manage multiplayer games. Also supports remote control of robots connected to other clients. TCP connections are used for communication and in order delivery is guarunteed. A hub and spoke network is created with host at the center, if host leaves game is toast. Currently if any player leaves the game state will transition to RKMultiplayerGameStateEnded It is a singleton class and should not be allocated.
