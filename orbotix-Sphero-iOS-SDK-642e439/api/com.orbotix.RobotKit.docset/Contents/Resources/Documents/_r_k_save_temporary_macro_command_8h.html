enum RKMacroFlags

Enumerations for the various flags that can be set for a macro. Multiple flags can be set by using a bitwise or operation on them.

Enumerator:
RKMacroFlagNone 

Used when no flags are to be set.

RKMacroFlagMotorControl 

Kills the drive motors on exit or abort.

RKMacroFlagExclusiveDrive 

Gives the macro exclusive control of driving.

RKMacroFlagUseVersion3 

Execute macro as a version 3 macro.

RKMacroFlagInhibitIfConnected 

Inhibit execution of this macro if a smartphone client is connected.

RKMacroFlagEndMarker 

Emit a macro marker with marker id 0 as macro exits.

RKMacroFlagStealth 

Macro execution does not reset the sleep timer.

RKMacroFlagUnkillable 

This macro can not be aborted. This is only valid for system macros.

RKMacroFlagExtendedFlags 

Indicates there are extended flags.
